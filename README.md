# HuskyFollow
- 使用方法
 
> 进入工作目录的src目录
  ```
  $ roscd
  $ cd ../src
  ```
> 克隆代码
  ```
  $ git clone https://github.com/muyangren907/HuskyFollow.git
  ```
> 编译
  ```
  $ cd ..
  $ catkin_make
  ```
> 运行
  ```
  $ roslaunch husky_follow husky_follow.launch
  ```
